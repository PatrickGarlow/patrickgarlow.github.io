Snake

Language Used: Python

Project Description:

This is a recreation of the class game "Snake" using python and pygame. Not only can the snake eat the food, but also a powerup that doubles the food eaten and one that slows the game down for a short period of time.

Challenges:

This was my first time using the PyGame module. Learning how to draw the squares and the text to the window in a way that made everything look nice was a speedbump.
Another challenge was adding the different powerups. For the "slow" effect I slowed the game speed for a number of seconds.

Lessons Learned:

Adding features late in development can be challenging to work background past code.
