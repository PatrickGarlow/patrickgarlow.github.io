Enigma Encryption Machine (GUI)

Languages Used: Python

Project Description:

This is a recreation of the German Encryption machine from The second World War. I had already built this in Java only using the commandline for input, but as I was learning about User Interfaces I decided to make it again in python. I used PyQt5 for this project.

Challenges:

Having both a manual encruption, letter by letter, and automatic was a challenge at first. For the first part they operate differently. One letter versus many. I looped through each letter and then combined the encrypted output again. This saved me many lines of code that I would've used had I written two different but identical programs.

Lessons Learned:

I had build this in commandline my freshman year of college using Java. I learned that just bacause I had done that and knew the solution for that case, didn't mean it would translate to this new language and using a GUI. I went back to basics of drawing out how the device worked and built up the python from there.
